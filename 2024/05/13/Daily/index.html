每日任务

发表于2024-05-13|更新于2024-05-20|生活
|阅读量:

前言

在本学期尝试了比较长时间的近高中作息后，学习成绩确实有一定长进，然而依旧并不理想。海量的学习时间并没有带来可观的回报。我知道，这明显是高强度水群水朵朵水98摸鱼导致的。尽管学习时我不会刷抖音打游戏，但是这些碎片时间中的碎片时间堆积起来，仍然不可小觑。但是不摸鱼是不可能的，为了保证我每天的卷生卷死都劳有所得，我写了这篇长期blog来记录自己每天的工作量。

5月13日

  1. 复习大雾第六章（驻波以前部分）
  2. 背诵大英U5
  3. 规划考拉工作室技术部团建日程
  4. 大雾作业（一半）

周一只有一节大雾，和大写一节水课，如此丰裕的时间，不仅没有玩到，还就写了这么点东西。效率这么低下还不如去打游戏呢。由此观之，衡量卷的从来不是量，而是质。如果可以在保持当前卷的时间的基础上，提升效率，那一定大有裨益。这天的工作量并不是规划，只是回顾，往后每天我都会给自己布置适当的大额工作量，防止自己假努力。

5月14日

由于明天就要考大雾小测，这天侧重点主要还是偏向于大雾。应当尽早完成其他任务后再复习大雾。

  1. 复习大雾第六章（驻波及以后）
  2. 背诵大英U5、U6
  3. 填报主题三终赛申报表

课太多了……不过好在大雾学的还好，虽然求不来满分，但总比之前强。大概。另外，由于各种原因，技术部的团建被考拉全体团建吞掉了，这样也好，可以升级为轰趴，不过技术部的团建多半要推迟到下学期了。说不定可以在招新前聚一次，招新后再聚一次？反正到时候我是部长我说了算。gingo说他为了srtp睡在bgs里。我在想，机器人基地那个工位应该是可以刷夜的，我要不要买把摇椅，也睡在里面？那里离东教挺近，上课也方便，不过我不想吃麦斯威。

5月15日

大雾测完了，应该可以准备vjf期末考了。

  1. 完成vjf作业
  2. 完成大雾作业
  3. 过一遍羽毛球理论考

因为明日方舟更新和美化网站的失败等问题，今天基本啥也没干，就当休假了。

5月16日

至暗的一天。

  1. 确认部长职务
  2. 写了部分第二类曲线积分的讲义
  3. 写vjf作业

啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊当部长真难！感觉自己不小心接过了很重的锅啊！

5月17日

周五，总算有可以自习的时间了。记得打印六级口试准考证。

  1. 写讲义
  2. 写vjf作业
  3. 写大雾作业
  4. 下午一点半 大语言模型机械臂项目 应该是我不参与 但要模仿学习如何启动项目
  5. 听大疆讲座3:00
  6. 过一遍羽毛球理论考
  7. 捋一遍部长职务

然而这天并没有进行很多的学业复习，晚上还打了麻将。

5月18日

这个周末要做很多很多繁琐的事情，它们分散在相隔较近的时间，导致我原本大块大块的复习时间支离破碎，妈的。

  1. 羽毛球理论考14:00 海洋大楼1、2楼计算机房
  2. 支教面试20:15 白沙一舍党员之家
  3. 筹备技术部第二次例会
  4. 复习二重积分

没有做太多学习方面的事情。我突然意识到自己先前之所以能没完没了地卷生卷死，并不是因为我多有毅力，只是因为我闲的没事干。当今天我忙的不可开交的时候，我把余下来的时间吃了半个西瓜，明日事少，可缓缓卷也。

5月19日

要体测和口试。狗屎。

  1. 体测
  2. 六级口试9:30 东6-219
  3. 完成一张vjf卷子
  4. 开技术部第二次例会
  5. 修改西方美学导论中期论文

第一次组织例会，打算整理个简单的小讲稿，不至于卡壳。

5月20日

大雾考的奇差无比，我几乎要怀疑自己内卷的意义了。。

  1. 复习大雾
  2. 复习大英
  3. 复习二重积分、偏导数
